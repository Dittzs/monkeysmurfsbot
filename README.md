# 🤖 Bot de Automação de Compras e Vendas — Discord

Bot desenvolvido em Python para automatizar fluxos de compra e venda de serviços digitais dentro de servidores Discord. O projeto simula um ambiente real de e-commerce, com controle de transações, comandos interativos e respostas automáticas.

---

## ⚙️ Funcionalidades

- Cadastro e listagem de serviços digitais disponíveis
- Fluxo completo de compra: seleção, confirmação e registro da transação
- Fluxo de venda: publicação de serviços por usuários autorizados
- Respostas automáticas via comandos customizados no Discord
- Registro e histórico de transações em tempo real
- Controle de permissões por cargo dentro do servidor

---

## 🛠️ Tecnologias utilizadas

- **Python 3** — lógica de negócio e automação
- **discord.py** — integração com a API do Discord
- **JSON / SQLite** — persistência e registro de dados
- **Git & GitHub** — controle de versão
